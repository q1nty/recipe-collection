# Архитектура проекта

## Используемые технологии

- Python 3
- Flask
- SQLite
- HTML
- CSS
- Pytest
- GitHub

## Схема работы

Пользователь → Браузер → Flask → SQLite

## Основные маршруты

/ - список рецептов

/add - добавление рецепта

/recipe/<id> - просмотр рецепта

/edit/<id> - редактирование

/delete/<id> - удаление
