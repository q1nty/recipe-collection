
## 1. Проблема

Многие пользователи хранят рецепты в разных местах: блокнотах, заметках телефона, сообщениях или браузерных закладках. Из-за этого нужный рецепт сложно быстро найти. Приложение позволяет хранить все рецепты в одном месте и быстро получать к ним доступ.

## 2. Целевая аудитория

* Студенты
* Любители готовить
* Домохозяйки
* Пользователи, которые хотят хранить собственную коллекцию рецептов

## 3. Функциональные требования

1. Добавление нового рецепта.
2. Просмотр списка всех рецептов.
3. Просмотр детальной информации о рецепте.
4. Редактирование существующего рецепта.
5. Удаление рецепта.
6. Поиск рецептов по названию.
7. Фильтрация рецептов по категории.

## 4. Данные рецепта

Каждый рецепт содержит:

* Название
* Категорию
* Ингредиенты
* Способ приготовления
* Время приготовления

## 5. Архитектура

Браузер → Flask → SQLite

Основные маршруты:

* `/` — список рецептов
* `/add` — добавление рецепта
* `/recipe/<id>` — просмотр рецепта
* `/edit/<id>` — редактирование рецепта
* `/delete/<id>` — удаление рецепта

## 6. Используемые технологии
