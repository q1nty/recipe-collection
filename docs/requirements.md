# Требования к проекту «Коллекция рецептов»

## Проблема
Пользователи хранят рецепты в разных местах, поэтому нужный рецепт бывает сложно найти.

## Целевая аудитория
- Студенты
- Любители готовить
- Домохозяйки

## Функциональные требования
1. Добавление рецепта
2. Просмотр всех рецептов
3. Просмотр одного рецепта
4. Редактирование рецепта
5. Удаление рецепта
6. Поиск по названию
7. Фильтр по категории

## Архитектура

Браузер → Flask → SQLite

Маршруты:
- /
- /add
- /recipe/<id>
- /edit/<id>
- /delete/<id>
